# Test Corpus

Create a private corpus with representative files before implementing real conversion:

- Simple text PDF
- Multi-column PDF
- PDF with tables
- PDF with images and captions
- Scanned PDF
- OCR PDF with poor text layer
- Very large PDF, 500+ pages
- Simple PPTX
- PPTX with speaker notes
- PPTX with charts
- PPTX with screenshots
- PPTX with embedded fonts
- Corrupt PDF
- Password-protected PDF
- Unsupported file renamed as `.pdf`

Do not commit customer documents or sensitive files to this repository.
